MeraJob Platform

AI-Powered Recruitment & Collaboration System

MeraJob Platform is an integrated recruitment ecosystem designed to connect employers, consultants, and candidates through AI-driven candidate-job matching, automated communication, and private talent pool management. It simplifies recruitment workflows, enabling end-to-end hiring collaboration.

Objective

To build a robust recruitment platform that automates hiring processes through intelligent matching, contextual communication, and centralized collaboration between employers, consultants, and candidates.

Key Features

  • Private candidate and consultant pools with permission-based access.
  • Advanced ATS (Application Tracking System) for tracking candidate progress.
  • Event-based Communication Engine (CIM) to automate emails, SMS, and notifications using parametric templates.
  • AI-driven Job-Candidate Matching using both parameter-based and semantic search logic.
  • Detailed analytics and candidate pipeline reporting tools.
  • Integrations with external APIs – Rchilli Parser, Oust Labs, and social OAuth logins.

Technology Stack

Layer Technology Purpose
Backend PHP (LAMP Stack), Python Core logic, semantic matching, and service orchestration
Frontend HTML, CSS, JavaScript, jQuery Interactive and responsive user interface
Database MySQL, Cassandra Relational and distributed data handling
Search Engine Apache Solr Parameter-based job and candidate discovery
Cloud Infrastructure AWS, Azure High-availability and secure deployment
Communication Amazon SES, VivaConnect, SMSLane Automated emails, SMS, and notifications
Version Control SVN, AWS S3 Source management and automated backups

System Modules

  • Employer Module: Job posting, private pool creation, candidate allocation.
  • Consultant Module: Candidate sourcing, job sharing, and candidate remarks.
  • Candidate Module: Profile updates, direct applications, and live job recommendations.
  • Communication Intelligence Module (CIM): Event triggers for templates and reply tracking.
  • ATS Module: Workflow automation for candidate movement and performance reporting.
