Project Overview

SourceInfi (by Shifto Pvt Ltd) is a dropshipping and e-commerce platform designed to simplify product listing, pricing, and order management for sellers. The platform connects sellers to marketplaces, provides fast Shopify integrations, and links multiple courier partners for seamless fulfillment.

Responsibilities are split across technologies: seller and transactional logic runs on PHP + MySQL, while courier API orchestration and real-time event handling run on Node.js + MongoDB. RabbitMQ orchestrates asynchronous jobs and retries; AWS hosts infrastructure and storage.

Key Modules

  • Seller & Vendor Registration — KYC, onboarding, and multi-tenant seller profiles.
  • Catalog Management — product listings, images, variant mapping, and pricing rules.
  • Shopify App Integration — fast product & order sync, webhook handling, app auth.
  • Pincode Management — serviceability and delivery SLAs per pincode.
  • Shipment Process — label generation, courier selection, and batch processing.
  • Shipment Tracking — aggregated courier status, webhooks, and fallback polling.
  • NDR Handling — automated non-delivery workflows and rescheduling.
  • Wallet & Margin Management — seller wallets, margin calculations, and settlement flows.

Technical Highlights

  • Hybrid architecture: monolithic PHP for seller ops and performant Node.js microservices for courier orchestration.
  • Event-driven processing with RabbitMQ for retries and throughput resilience.
  • Shopify app that supports OAuth flows, webhooks and bulk sync.
  • Optimized MySQL schemas for catalog and wallet ledgers; MongoDB for courier events and fast lookups.
  • Infrastructure on AWS with autoscaling, S3 for assets, and CloudWatch for monitoring.
